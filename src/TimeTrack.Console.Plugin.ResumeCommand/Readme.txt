This plugin will start tracking with the identifier selected from the saved activities/identifiers, according to the following commands.
If no record can be retrieved by the command (e.g. selecion criteria is invalid etc.), tracking is not going to be started!

Available Commands:
- Positive Integer (e.g. 0, 1, 2 etc)
If currently tracking		=> Skip the x amount of records in the tracked activities and select the next
If currently NOT tracking	=> Skip the x amount of records in the tracked activities and select the next

- last
If currently tracking		=> Last tracked activity
If currently NOT tracking	=> Second before last tracked activity
